agent-first path
No browser required

        
The probe must execute where the agent runs. A hosted MCP gateway can only measure itself.

    

    

        
01
Create manifest
POST /v1/ip-check/sessions returns dual-stack HTTPS targets, a unique DNS name, STUN targets, an identity challenge, and an authenticated report URL.

        
02
Probe locally
Use the Python client’s network_environment_check() or the local MCP tool of the same name. Traffic leaves the runtime under test.

        
03
Read assurance
Every result says whether it was server-observed, client-declared, signature-verified, or attested. Model names remain claims unless independently attested.

    

    
async with HyruleClient("https://cloud.hyrule.host") as hc:
    result = await hc.network_environment_check(
        expected_dns_resolvers=["1.1.1.0/24"],
        protocol="sdk",
    )






    
interpretation limits
Evidence, not an evasion score.

    
DNS resolver differences are not automatically leaks. Blocked WebRTC or failed STUN is inconclusive. A fingerprint is not identity. Hyrule does not claim that these signals can bypass another platform’s abuse controls, and does not infer a model vendor from writing style.
