a. direct API

402 is the quote.

Discovery prices are estimates. The unpaid request returns the exact base-unit amount, accepted networks, asset, and payee. Sign one acceptance and retry the identical request.

Headless golden path

1 · discoverGET /openapi.json
2 · requestPOST /v1/dns/lookup
3 · quote402 Payment-Required
4 · settlePayment-Signature: eyJ…
5 · consume200 / 202 / binary
# request without credentials
curl -i https://cloud.hyrule.host/v1/dns/lookup \
  -H 'content-type: application/json' \
  -d '{"name":"example.com","type":"AAAA"}'

# sign accept.amount exactly, then replay
curl https://cloud.hyrule.host/v1/dns/lookup \
  -H 'Payment-Signature: eyJ…' \
  -H 'content-type: application/json' \
  -d '{"name":"example.com","type":"AAAA"}'
b. browser agents

WebMCP keeps the work in the page.

When the browser supports WebMCP, the toolbox exposes a small set of state-aware tools. An agent can use the installed wallet or provide a payment payload signed by its own wallet. Hyrule adds no spending cap or confirmation step to agent calls.
toolpurpose
search_hyrule_diagnosticsSearch the currently enabled diagnostic catalog and update the visible selection.
quote_hyrule_diagnosticValidate input and return an ephemeral request-bound x402 quote.
pay_hyrule_diagnosticSign with the browser wallet or submit an agent-signed x402 payload.
get_hyrule_diagnostic_resultWait for jobs, page through sanitized results, and complete downloads.
WebMCP is progressive enhancement. Agents without a visible supported browser use the direct API path above.
